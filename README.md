# nomos-protocol

Governance protocol for autonomous AI agents: evidence, decisions, receipts and replayable control.

## Overview

This repository defines the core concepts, schemas, decision records, and governance primitives for the NOMOS protocol.

- Jurisdiction schemas (EU, JP, IN, KR, SG, CN, BR, NG, ZA, MX)
- Decision record format
- Evidence receipt spec
- Preflight verdict types: `ALLOW` / `ALLOW_PUBLIC_SIGNAL_ONLY` / `REVIEW` / `BLOCK`
